# Outputs Directory

This directory contains visualizations and results generated from the Movie Recommendation System analysis.

## Contents

- `rating_distribution.png` - Distribution of user ratings across the dataset
- `genre_analysis.png` - Analysis of movie genre popularity
- `top_rated_movies.png` - Visualization of most rated and highest rated movies
- `user_activity.png` - Distribution of user engagement metrics
- `model_evaluation.png` - Model performance visualizations (predicted vs actual, residuals)

## Note

These visualizations are generated automatically when running the Jupyter notebook. The images are embedded in the main README.md file for easy viewing.
